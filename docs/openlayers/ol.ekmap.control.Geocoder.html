Class: Geocoder

Geocoder

Geocode.

new ol.ekmap.control.Geocoder(target, options)

control/Geocoder.js, line 6

Control tìm kiếm vị trí theo địa chỉ, với nhiều nguồn dữ liệu: OSM, Bing, Mapquest, Opencage, Photon.

Không sử dụng trên bản đồ

Name Type Description
target Object 可选

Specify a target if you want the control to be rendered outside of the map's viewport.
If target is equal to null or undefined, control will render by default.
options Object 可选
